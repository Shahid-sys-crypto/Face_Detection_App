# 🎯 Face Detection App

A simple Face Detection application built using **OpenCV** and **Tkinter** in Python.  
This app allows users to detect faces either from an image file or through a live webcam feed.

---

## 📌 Features

- Detect faces from an image  
- Real-time face detection using webcam  
- Simple GUI using Tkinter  
- Uses Haar Cascade Classifier from OpenCV  

---

## 🛠️ Technologies Used

- Python  
- OpenCV (`cv2`)  
- Tkinter
